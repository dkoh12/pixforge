# pixforge GUI — Plan

## Goal
Add a local web-based GUI to pixforge with drag & drop image conversion that auto-downloads the result to the user's Downloads folder.

## Tech Stack
- **Backend**: Flask (lightweight, simple, no overhead)
- **Frontend**: Plain HTML/CSS/JS — no framework needed
- **Integration**: Reuses existing `pixforge.converter` and `pixforge.transforms` modules

## Features
- Drag & drop (or click to browse) image upload
- Select output format (PNG, JPEG, WebP, AVIF, GIF, TIFF, BMP)
- Options: quality, scale %, width, height, DPI, rotate, flip, grayscale
- Convert button → processed image auto-downloads to browser's Downloads folder
- Preview original image before converting
- Show file info (dimensions, size, format)

## File Structure
```
pixforge/
├── gui/
│   ├── __init__.py
│   ├── app.py          # Flask app
│   └── templates/
│       └── index.html  # Single page UI
```

## CLI Command
```bash
pixforge gui           # starts server at http://localhost:5000
pixforge gui --port 8080
```

## Flow
1. User drags image onto page → preview shown
2. User picks format + options
3. User clicks "Convert"
4. Browser POSTs image to Flask → pixforge processes it → returns converted file
5. Browser triggers automatic download → file lands in ~/Downloads

## Notes
- Auto-download works via browser's native download behavior (Content-Disposition: attachment)
- No files stored on disk — process in memory using BytesIO
- Add `flask` to dependencies in pyproject.toml
